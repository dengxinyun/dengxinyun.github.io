vue

  • 样式
    行内样式背景图片的写法：
    :style=”{backgroundImage:’url(‘ + imgURL + posterInfo.qrcode_bg + ‘)’,’min-height’:wH+’px’}”

  • 轮播
    mint-ui组件，轮播图是用定位的方式排版的，需要设定一个高度。可以额外引入一个轮播插件：推荐vue-swiper-component
    参照地址：https://www.cnblogs.com/qq735675958/p/8423277.html

    路由懒加载就是在配置路由的时候将component配置成 ：
    component: resolve => require([‘components/About.vue’], resolve)

  • 缓存、刷新
    vue使用keep-alive实现数据缓存不刷新，主要是在入口APP.vue里面实现，
    献上地址：https://segmentfault.com/a/1190000011640453

  • 缓存、位置
    vue-router从详情页返回列表页时返回到上次滚动位置：结合上面的keep-alive,用下面的scrollBehavior方法

    1.router文件中设置为 mode: ‘history’, 模式

    2.router设置:

    scrollBehavior (to, from, savedPosition) {
        if (savedPosition) {
          return savedPosition
        } else {
          return { x: 0, y: 0 }
        }
    }
